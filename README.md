# ⚡ Ankit Gupta — Engineering Portfolio

Live: https://ankit-gupta-engineering-portfolio.vercel.app
Repo: https://github.com/Ankit22410/ankit-gupta-engineering-portfolio

## Tech Stack
React 18 · Three.js r128 · Vite 5 · Vercel · Google Analytics 4

## Features
- Three.js 3D interactive drone + orbiting satellite (mouse-reactive)
- Animated system boot loader with module progress
- Orbital skill constellation canvas animation
- Live engineering metric counters (IntersectionObserver)
- Full SEO: meta · Open Graph · Twitter Card · JSON-LD Person schema
- Google Analytics 4 page tracking
- Security headers A+ via vercel.json
- Code-split bundles (react-vendor, three-vendor) + Terser minification

## Local Development
  npm install
  npm run dev        # http://localhost:3000

## Production Build
  npm run build      # outputs to dist/
  npm run preview    # preview the build locally

## Deploy to Vercel
  npm i -g vercel
  vercel login
  vercel --prod

## Assets to add (public/assets/)
  resume.pdf         Your CV — linked from the portfolio
  og-image.png       1200x630 social share preview image
  profile.jpg        Your photo (optional)

## Environment Variables
  VITE_GA_ID         Your GA4 Measurement ID (e.g. G-XXXXXXXXXX)
